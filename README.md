# aws-cloudformation-yaml

A collection of reusable **AWS CloudFormation templates written in YAML** for provisioning and managing cloud infrastructure efficiently and reliably.

## Features

- Modular and well-structured templates
- Supports common AWS resources: VPC, EC2, S3, IAM, RDS, Lambda, API Gateway, and more
- Follows Infrastructure as Code (IaC) best practices
- Easy to customize and extend for different environments (dev, staging, prod)
- Includes example stacks for quick deployment

## Prerequisites

- [AWS CLI](https://aws.amazon.com/cli/) installed and configured
- AWS account with necessary IAM permissions
- Basic understanding of CloudFormation, YAML, and AWS services
